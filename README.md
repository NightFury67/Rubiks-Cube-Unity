# Rubik's Cube Game

A 3D Rubik's Cube game built in Unity 3D.

## Controls
- U / Y — Top face clockwise / counter-clockwise
- R / E — Right face clockwise / counter-clockwise  
- L / K — Left face clockwise / counter-clockwise
- Right Mouse Button + Swipe — Rotate entire cube

## Features
- Fully functional 3D Rubik's Cube
- Smooth face rotation animations
- Shuffle button
- Whole cube rotation via mouse swipe
